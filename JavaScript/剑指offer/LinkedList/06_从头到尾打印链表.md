### [剑指 Offer 06\. 从尾到头打印链表](https://leetcode-cn.com/problems/cong-wei-dao-tou-da-yin-lian-biao-lcof/)

Difficulty: **简单**


输入一个链表的头节点，从尾到头反过来返回每个节点的值（用数组返回）。

**示例 1：**

```
输入：head = [1,3,2]
输出：[2,3,1]
```

**限制：**

`0 <= 链表长度 <= 10000`


#### Solution

Language: **JavaScript**

```JavaScript
// Iteration
​   var reversePrint = function(head) {
        let arr = [];
        if (!head) {
            return arr;
        }

        while(head) {
            arr.push(head.val);
            head = head.next
        }
        return arr.reverse();
    };

// Recursion 利用递归栈的特性，先进后出得到反转数据
    var reversePrint = function(head) {
        if (!head) return []
        let arr = reversePrint(head.next) // 剩下的反转打印
        arr.push(head.val);
        return arr
    };
```
