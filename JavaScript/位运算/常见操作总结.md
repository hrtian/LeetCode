## 常见二进制操作
### 1. 基本操作
```JavaScript
    a = 0 ^ a
    0 = a ^ a
    // 推导出：
    a = a ^ b ^ b
```

### 2. 判断奇偶性
```JavaScript
    // 奇数
    a & 1 === 1
```

### 3. 交换两个数
```JavaScript
    a ^= b
    b ^= a
    a ^= b
```

### 4. 移除最后一个 1
```JavaScript
    a = n & (n-1)
```

### 5. 获取最后一个 1
```JavaScript
    diff = (n & (n-1) ) ^n
```

### 6. 变换符号
```JavaScript
    ~a + 1
```

### 7.取余操作
```javascript
    // a & (b -1)
```

### 8.取int型变量的第K位
```javascript
    a >> (k - 1) & 1
```

### 9.取末K位
```javascript
    a & ((1 << k) - 1)
```
